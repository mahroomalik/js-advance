DOM stands for Document Object Model. It is a programming interface that allows us to create, change, or remove elements from the document. In HTML, ids are used as unique identifiers for the HTML elements. This means you cannot have the same id name for two different elements. We can also add events to these elements to make our page more dynamic. The DOM views an HTML document as a tree of nodes. A node represents an HTML element
